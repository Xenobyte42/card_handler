# CardSeacrcher

API for searching cards from Scryfall Magic Card searching

# Dependencies
- python3.5.x and more
- configparser
- aiohttp
- aioredis
- aiohttp_auth
- aiohttp_session
- aiohttp_jinja2
- jinja2
- urllib

# Usage

To start the web server for the first time, configure `config-file` redis.cfg:

`host`=database_hostname

`port`=database_port

`db`=database_number
